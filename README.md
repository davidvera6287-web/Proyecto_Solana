📝 Mi Lista de Tareas en Solana

Un CRUD on-chain para administrar tareas en la blockchain de Solana.
Desarrollado con Rust + Anchor desde Solana Playground.

---

¿Qué puede hacer?

Instrucción| Descripción
"crear_usuario"| Crea tu perfil personal (PDA)
"agregar_tarea"| Agrega una nueva tarea
"completar_tarea"| Marca tarea como completada
"eliminar_tarea"| Elimina una tarea
"ver_tareas"| Consulta tareas registradas

---

¿Cómo usarlo en Solana Playground?

Importa el proyecto

Abre Solana Playground y pega la URL del repositorio.

Conéctate a Devnet

Haz clic en Not Connected y conecta wallet.

Haz Build

Compila el programa.

Haz Deploy

Despliega el contrato en Devnet.

Corre los Tests

Ejecuta pruebas CRUD.
